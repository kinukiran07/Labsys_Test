Follow the Instructions to create Plugin Template

1.Update SVN_LabSys Folder for latest version
2.Go to SVN_LabSys\LabSys_SourceCode\Plugin Template\VIPM Source Files and double click on bsh_lib_plugin_template 
3.Install Package
4.Restart LabVIEW
5.Open LabVIEW and click on Create Project
6.Select the Plugin Template and Click on Next Button
7.Select the Project path and change the Project Name and click on Finish Button
